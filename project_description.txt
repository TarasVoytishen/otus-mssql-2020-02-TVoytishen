Проект Пифагор

Используется связка MSSQL-1C-Apache

на 1С опубликованы 2 rest сервиса getData и sendData  (работают через apache)

getData - запрос на получение информации из MSSQL базы в json

sendData - запись информации в MSSQL базу в json

Все нужно для того чтобы хранить разнородные данные,
которые поступают из разных источников

В 1С есть описание таблиц, источников данных, генерация хранимых процедур для создания таблиц, записи, удаления
